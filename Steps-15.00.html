Creating Visual - Cards 15

In this lesson, you will create visual cards for "Revenue Per Month", "Year over Year", "Profit Margin"

First KPI Card  "Revenue Per Month"

Step 1 - Open Visualization Pane

Step 2 - Select Visual Card

Step 3 - Drag the visual to the dashboard.

Step 4 - Click on Total Revenue Measure and drag it into the fields.

General: X - Position = 2, Y - Position = 247, Width = 207, Height = 96

Data Label: Color = #0D6ABF,  Value Decimal = 0, Font Size = 32,  Font type = DIN

Category = OFF

Title = ON,  Text Title = "Revenue Per Month", Font Color = #000000 , Background Color = #cccccc ,

Font Size = 14 and Font Family = DIN, Alignment = Center

Background ON, Color = #ffffff, Transparency = 0

Turn "Shadow" ON  Color = #094780

Step 5 - Grap "Total Revenue" measure from your "Key Measure" table.

Step 6 - Drop the "Total Revenue" measure under the "Field" 

Step 7 - Select "Paint Brush"  icon and makesure the settings are as shown below
