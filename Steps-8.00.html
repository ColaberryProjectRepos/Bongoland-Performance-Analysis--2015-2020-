Load CSV file into Power BI Desktop 8

Steps

Ø  Download Data file (zip) into your computer by clicking the red  icon above

Ø  Open Power BI Desktop

Ø  Click Get data on Power BI Desktop

Ø  Under Get Data select Text/CSV

Ø  Select data file you have downloaded - it contains 8 CSV files, uploads one file at a time

Ø  Click on the Load button to load the data
